# 闭包
	一个函数内嵌套一个子函数,子函数可以访问父函数的局部作用域
	一个函数调用另一个函数的局部变量
	function fn(){
		let a = 1;
		function fn2(){
			a++;
			console.log(a);
		}
		fn2();
	}
	fn();

# 自执行函数--在函数后面直接加()会有语法错误
	(function (index){
		console.log(index);
	})(i)


1> 利用了一个闭包 保存了一个局部作用域变量
2> 以函数参数形式传递的,
3> 每次循环生成的都是一个新的作用域执行环境
4> 函数作用域: 会将作用域下的局部变量 函数 存到一个单独的空间下
